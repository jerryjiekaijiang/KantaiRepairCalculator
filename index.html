1. Description

This program sorts a list of ships into a user defined number of docks and attempts to evenly distribute the repair times to find the most time efficient distribution of ships for repair. This does not take into consideration Akashi repair nor bucket usage, but recommendation will be added in the future.

2. Usage:

KC3 extension for chrome is currently required for this.

  1. Extract contents of the zip file into a folder.
  2. Place or create a .txt file in the same folder as this exe.
  3. Go to KC3 Strategy Room > Ships > Docking
  4. Copy and paste the ships in this section to your .txt file. For an example of what the result should look like, refer to repairsample.txt.
  5. Run the .exe and type in the name of your file including the ".txt" part.

3. Notes:

Currently unfinished as of right now, planning to add 3/4 dock options and an option to specify how many times maximum you want to check back into repairing.
